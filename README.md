# Weather Information Dashboard

A comprehensive React application built to display real-time weather details based on user input.

## Features & Implementation
* **API Integration:** Fetches accurate weather details dynamically from the OpenWeatherMap API.
* **State Management:** Manages weather data, loading states, and error handling seamlessly.

## Use of `useEffect` Hook
In this project, the `useEffect` hook is utilized to manage side-effects effectively:
* It triggers data fetching operations whenever the target city changes.
* It ensures clean and controlled asynchronous operations to prevent unnecessary API re-calls and optimized rendering.
